# Trident Web

This is a WIP [Next.js](https://nextjs.org) project made for the [Trident](https://github.com/pe3ep/Trident) MCCI mod

## Running locally

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
